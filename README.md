## Deployment Options

### Option 1: Local Viewing
Simply open `index.html` in your browser

### Option 2: Free Hosting (Recommended)
1. **Netlify** (netlify.com):
   - Drag and drop the entire `drtroy-website` folder
   - Instant live URL
   - Free custom domain support

2. **Vercel** (vercel.com):
   - Connect GitHub repo or drag/drop files
   - Automatic deployments
   - Fast global CDN

3. **GitHub Pages**:
   - Upload to GitHub repository
   - Enable Pages in settings
   - Free .github.io domain

### Option 3: Traditional Web Hosting
Upload all files to your web hosting provider's public folder (usually `public_html` or `www`)

## Customization

### Colors (CSS Variables in styles.css)
- **Primary Blue**: `--primary-blue: #1e40af`
- **Healthcare Teal**: `--healthcare-teal: #0891b2`
- **Success Green**: `--success-green: #059669`

### Content Updates
- Edit text directly in `index.html`
- Course information in the courses section
- Testimonials in the testimonials section
- Pricing information in the pricing section

### Adding New Pages
1. Create new HTML files following the same structure
2. Update navigation links in `index.html`
3. Maintain consistent styling with existing CSS
